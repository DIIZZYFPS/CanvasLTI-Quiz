CANVAS QTI CONVERTER - FORMATTING GUIDE

This guide explains how to format your quiz questions so they can be automatically converted into a Canvas QTI import package. 

*** GENERAL RULES ***
1. Separate Questions: Leave at least one blank line between each question block.
2. Points (Optional): You can specify point values anywhere in the question text. 
   Supported formats: "(1 point)", "(5 pts)", "[Points: 10]", "Points: 5", "Score: 2".
   Default: 1 point if not specified.
3. Validation: The tool will check your syntax. If you forget an "Answer:" line or have missing options, it will flag the error.

*** QUESTION TYPES ***

1. MULTIPLE CHOICE
   - Usage: Standard questions with one correct option.
   - Requirement: Question text, followed by options (A, B, C...), followed by "Answer: [Letter]".
   
   Example:
   Which programming language is known as the language of the web?
   A) Python
   B) JavaScript
   C) C++
   Answer: B

2. MULTIPLE ANSWERS (NEW)
   - Usage: Questions where students select more than one correct option.
   - Requirement: List multiple letters in the answer line OR mark correct choices with an asterisk (*).
   
   Example A (Starred):
   Select the prime numbers:
   *A) 2
    B) 4
   *C) 7
   
   Example B (Tagged):
   Select the prime numbers:
   A) 2
   B) 4
   C) 7
   Answer: A, C

3. TRUE/FALSE
   - Usage: Binary True/False choice.
   - Requirement: "Answer: True" or "Answer: False".
   
   Example:
   Water boils at 100 degrees Celsius. (1 point)
   Answer: True

4. SHORT ANSWER
   - Usage: Questions requiring a typed word or phrase match.
   - Requirement: Must have "Answer: [text]".
   
   Example:
   What is the capital of France?
   Answer: Paris

5. FILL IN MULTIPLE BLANKS (FMB) (NEW)
   - Usage: Questions with multiple blanks in one sentence.
   - Style A (Auto-Blank): Simply wrap the answers in brackets.
     Example: The capital of [France] is [Paris].
     
   - Style B (Mapped): Use bracketed variables and a mapping line.
     Example: The [c] of France is [p].
     Answers: c: capital, p: Paris

6. ESSAY
   - Usage: Open-ended text response (manual grading).
   - Requirement: Start with "Essay:" OR end with "[Essay]".
   
   Example:
   Essay: Discuss the socio-economic impacts of the Industrial Revolution.
   Points: 10

*** RESPONDUS COMPATIBILITY ***
The tool also natively supports the standard Respondus Legacy format (Type: MC, MR, F, FMB, E). You can paste raw Respondus exports directly into the tool.
