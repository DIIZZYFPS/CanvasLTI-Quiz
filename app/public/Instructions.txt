CANVAS QTI CONVERTER - FORMATTING GUIDE

This guide explains how to format your quiz questions so they can be automatically converted into a Canvas QTI import package. 

*** GENERAL RULES ***
1. Separate Questions: Leave at least one blank line between each question block.
2. Points (Optional): You can specify point values anywhere in the question text. 
   Supported formats: "(1 point)", "(5 pts)", "[Points: 10]", "Points: 5", "Score: 2".
   Default: 1 point if not specified.
3. Validation: The tool will check your syntax. If you forget an "Answer:" line or have missing options, it will flag the error.

*** QUESTION TYPES ***

1. MULTIPLE CHOICE
   - Usage: Standard questions with options.
   - Requirement: Question text, followed by options (A, B, C...), followed by "Answer: [Letter]".
   
   Example:
   Which programming language is known as the language of the web? (2 pts)
   A) Python
   B) JavaScript
   C) C++
   D) Java
   Answer: B

2. TRUE/FALSE
   - Usage: Binary True/False choice.
   - Requirement: Start with "TF:" OR end with "Answer: True" or "Answer: False".
   
   Example 1 (Standard):
   TF: The sun rises in the west. (1 point)
   Answer: False

   Example 2 (Compact):
   Water boils at 100 degrees Celsius. (T/F) Answer: True

3. SHORT ANSWER
   - Usage: Questions requiring a typed word or phrase match.
   - Requirement: Start with "SA:" OR end with "[Short Answer]". Must have "Answer: [text]".
   
   Example:
   SA: What is the capital of France? (2 points)
   Answer: Paris

4. FILL IN THE BLANK
   - Usage: Sentence completion.
   - Requirement: Use underscores (at least 2, e.g., "_____") to mark the blank.
   
   Example:
   The chemical symbol for water is _____. (3 pts)
   Answer: H2O

5. ESSAY
   - Usage: Open-ended text response (manual grading).
   - Requirement: Start with "Essay:" OR end with "[Essay]".
   
   Example:
   Essay: Discuss the socio-economic impacts of the Industrial Revolution.
   Points: 10
